**Windows-Firewall-IP-Blocking**

**Objective** 
- Block a specific IP address using Windows Firewall.
  
**Tools Used**

Windows Defender Firewall

**Steps Performed**

- Opened Windows Defender Firewall.
- Created a new inbound rule.
- Selected Custom Rule.
- Specified the target IP address.
- Configured the action to Block.
- Applied and verified the rule.
  
**Skills Demonstrated**

- Host-Based Firewall Management
- Network Security
- Access Control

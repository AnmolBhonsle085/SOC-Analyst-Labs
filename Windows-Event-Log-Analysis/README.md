**Windows Event Log Analysis**

**Objective:**

- Analyze Windows Security and System logs using Event Viewer to identify authentication events, account activity, and system operations.

**Tools Used:**

- Windows Event Viewer

**Event IDs Reviewed:**

- 4624 – Successful logon
- 4625 – Failed logon
- 4672 - Special Privileges Assigned
- 4104 - PowerShell Script Block    Logging
- 4720 – User account created
- 4726 – User account deleted

**Steps Performed:**

- Opened Event Viewer and accessed Security and System logs.
Filtered events by Event ID.
- Reviewed successful login events (4624).
- Analyzed failed login attempts (4625).
- Examined user account management events.
- Reviewed system startup and shutdown records.
- Documented observations and event details.

**Findings:**

- Identified successful and failed authentication attempts through Event IDs 4624 and 4625.
- Observed user logon and logoff activity.
- Reviewed account management events and system operational logs.
- Developed familiarity with Windows event monitoring and security log analysis.

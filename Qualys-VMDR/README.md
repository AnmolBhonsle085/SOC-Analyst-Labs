# Qualys VMDR

## Objective
Explore Qualys VMDR Community Edition and understand the vulnerability management workflow by configuring assets and preparing an internal vulnerability scan.

## Tool Used
- Qualys VMDR Community Edition

## Activities Performed
- Created a Qualys Community Edition account.
- Explored the Qualys VMDR dashboard.
- Added my local Windows system as an asset using its private IP address.
- Configured a vulnerability scan using the default option profile.
- Attempted to launch an internal vulnerability scan.

## Observation
The scan could not be completed because no **Qualys Scanner Appliance** was configured. Internal vulnerability scans require a registered Scanner Appliance to scan private network assets.

## Skills Demonstrated
- Qualys VMDR
- Vulnerability Management
- Asset Management
- Scan Configuration
- Security Assessment
- Enterprise Vulnerability Management Workflow

## Learning Outcome
- Understood the Qualys VMDR interface and workflow.
- Learned how to add assets to a subscription.
- Learned how to configure a vulnerability scan.
- Understood the role of a Scanner Appliance in internal network scanning.
- Gained hands-on exposure to enterprise vulnerability management concepts.
